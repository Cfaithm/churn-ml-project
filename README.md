# Churn Prediction Project

Beginner-friendly machine learning project to predict customer churn using transactional and activity data.

## Project Goals
- Build a clean ML pipeline
- Prevent data leakage
- Train baseline models
- Evaluate performance honestly

## Dataset
Synthetic SaaS-style dataset including:
- customers
- subscriptions
- transactions
- user activity

## Models Used
- Logistic Regression
- Random Forest

## Tools
Python, Pandas, Scikit-learn, Jupyter Notebook
